# Discrete Mathematics Using Python

A collection of **Python-based Discrete Mathematics programs and notebooks** covering important concepts such as sets, relations, logic, graph theory, and algebraic structures.

---

## 📌 About the Repository

This repository contains Python implementations of various **Discrete Mathematics** topics.  
It is designed to help students understand mathematical concepts through code and practical notebook-based examples.

The repository currently includes work on topics like:

- Cartesian Product
- Cardinality of Sets
- Set Operations
- Cayley Table
- Equivalence Relations
- Graph using Adjacency Matrix
- Handshaking Lemma
- Logical Equivalence
- Truth Tables for Simple and Compound Propositions

These topics are reflected in the notebook files currently present in the repository. :contentReference[oaicite:1]{index=1}

---

## 🎯 Objectives

The main goals of this repository are:

- To apply **Python programming** in Discrete Mathematics
- To make abstract mathematical concepts easier to understand
- To provide notebook-based examples for study and practice
- To help students revise and experiment with core topics visually and logically
